# 基于PyTorch的神经网络对话模型

#### 介绍
一个简单的对话模型以及训练Demo。  
基于PyTorch框架搭建。  
使用transformer结构且基于全连接层的神经网络来实现。  

#### 结构 
data； 训练完成的模型存放位置  
moudel：ransformer结构的模型  
qadata：训练数据采用固定列表的形式，可以根据需求自行调整  
response：用于根据输入句子生成响应  
train：模型训练，学习率、早停耐心值可根据需求自行调整  

#### 说明
只是一个粗略的demo，支持cpu或者gpu，旨在初步学习了解。因为本人是java转，所以可能有很多错误的地方，望指正！
